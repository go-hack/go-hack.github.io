How does this work

This project is for anyone who wants to learn how to code, for those who want a refresher of the basics or anyone just hack some golang for fun.

Go hack will be a hobby course hosted on github and much will be done through it. We will focus on the basics of computer science in order to get a fundamental understanding of programming and we will do so through the programming language go. This means that if you want to learn go in particular, this might not be for you.

It is an ongoing project to add content, so new stuff will hopefully appear in due time. If you want us to discuss anything in particular, pleas let me know
